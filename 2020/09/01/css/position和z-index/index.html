position定位

position属性指定一个元素（静态的，相对的，绝对或固定）的定位方法的类型。

realtive

以自身为参照物移动到指定的位置，并且被定位的元素会占据原有的位置

<style>
		div{
			width: 200px;
			height: 200px;
        }
        .d1{
			background-color: red;
        }
        .d2{
			background-color: black;
			position: relative;
/* 			left: 200px;
			top: 200px; */
        }
        .d3{
			background-color: aliceblue;
        }
    </style>
    <body>
        <div class="d1"></div>
        <div class="d2"></div>
        <div class="d3"></div>
    </body>

absolute

以其他元素作为参照物，移动指定距离，且不会占据原有的位置，关于 absolute 定位参考点的选择：

  1. 如果元素的外层元素是非 static 定位（有除默认属性之外的定位属性），那么外层元素就成为了该元素的定位参考点。
  2. 如果元素的外层元素没有设置任何 position 的值，那么该元素将寻找离自己最近的设定过非默认 position 属性的外层元素作为参照物。
  3. 如果元素的外层元素没有找到设定为非默认 positon 属性的外层元素，那么此时定位参考元素变为 body ，或者说页面

z-index堆叠顺序

一旦修改了元素的定位方式，则元素可能会发生堆叠。可以使用z-index属性来控制定位元素的出现顺序。z-index仅能在定位的元素（使用了 position 非默认属性的元素）上生效
z-index属性：
值为数值，数值越大表示堆叠顺序越高，即离用户越近，可以设置为负值，表示离用户更远 ，一般不设置负值

<style>
        .d1{
			background-color: black;
			width: 200px;
			height: 200px;
			position: absolute;
			left: 200px;
			top: 200px;
        }
        .d2{
			position: fixed;
			background-color: red;
			left: 100px;
			top: 100px;
			width: 200px;
			height: 200px;
        }
    </style>
    <body>
        <div class="d1"></div>
        <div class="d2"></div>
    </body>

可以发现后面的div会覆盖前面的，如果想让前面的覆盖后面的，可以设置 z-index：

<style>
        .d1{
			background-color: black;
			width: 200px;
			height: 200px;
			position: absolute;
			left: 200px;
			top: 200px;
			z-index: 1;
        }
        .d2{
			position: fixed;
			background-color: red;
			left: 100px;
			top: 100px;
			width: 200px;
			height: 200px;
        }
    </style>
    <body>
        <div class="d1"></div>
        <div class="d2"></div>
    </body>
